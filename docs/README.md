# Project Documentation

> **For AI agents:** the authoritative routing index is in `/CLAUDE.md`.
> Do not read every file in this folder — most are historical or specialized.
> Use the index in CLAUDE.md to decide what's relevant to the current task.

## Structure

- `adr/` — Architecture Decision Records. Short markdown files capturing non-obvious decisions and tradeoffs. Use `adr/TEMPLATE.md` for new entries.
- `specs/` — Per-feature plans. Written before implementation; reviewed before code. Use `specs/TEMPLATE.md` for new entries.
- `runbooks/` — Incident response and operational procedures. Read at 2am.
- `archive/` — Superseded plans and old designs. Kept for reference; not authoritative.
